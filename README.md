# [千职鹤](https://www.careershe.com/)

## 关于千职鹤
「**[千职鹤](https://www.careershe.com/)**」 :fire:  是一个提供全面及准确的各种职业选择信息，帮助学生实现职业梦想的平台。把国外先进职业教育专业知识引入中国，而这些知识正是中国学生在学校里无法学到的。 :fire:
